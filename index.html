Question 2(a)(i) - Formulate the Relationship for a Three-Phase Half-Wave Converter

For a Three-phase, Half-wave, Two-Quadrant SCR bridge converter (as shown in Figure Q2), the output voltage is formed by three phase voltages ($V_a, V_b, V_c$) displaced by $120^\circ$ ($2\pi/3$ radians) from one another[cite: 1].

The Derivation Procedure:

The naturally occurring commutation point for a phase diode/SCR happens at $\omega t = 30^\circ$ ($\pi/6$). Therefore, the firing angle delay $\alpha$ is measured starting relative to this natural crossover point[cite: 1].

The conduction period for a single phase SCR spans a duration of $120^\circ$ ($2\pi/3$ radians), running from limits of $\omega t = \frac{\pi}{6} + \alpha$ up to $\omega t = \frac{5\pi}{6} + \alpha$[cite: 1].

Let the phase voltage be: v_an(wt) = V_ph_max * cos(wt) (or V_m sin(wt) relative to crossover) To calculate the average DC output voltage (V_d): V_d = (3 / 2π) * ∫π/6 + α5π/6 + α V_m * sin(wt) d(wt) V_d = (3 * V_m / 2π) * [ -cos(wt) ] from (π/6 + α) to (5π/6 + α) V_d = (3 * V_m / 2π) * [ cos(π/6 + α) - cos(5π/6 + α) ] Using trigonometric identity: cos(A) - cos(B) = 2 * sin((A+B)/2) * sin((B-A)/2) V_d = (3 * V_m / 2π) * [ 2 * sin(π/2 + α) * sin(π/3) ] Since sin(π/2 + α) = cos(α) and sin(π/3) = √3 / 2: V_d = (3 * V_m / 2π) * [ 2 * cos(α) * (√3 / 2) ] V_d = (3 * √3 * V_m / 2π) * cos(α) If expressed in terms of the RMS Line-to-Line Input Voltage (V_LL): Since V_m (peak phase voltage) = √2 * V_phase = √2 * (V_LL / √3) V_d = (3 * √3 / 2π) * (√2 * V_LL / √3) * cos(α) V_d = (3 * √2 / 2π) * V_LL * cos(α) ≈ 0.675 * V_LL * cos(α)

Key Properties for Assignment Presentation:

  • Two-Quadrant Operation: Because the current ($I_d$) can only flow in one direction through the SCRs, it stays positive[cite: 1]. However, by driving $\alpha > 90^\circ$, the average output voltage $V_d$ drops negative, moving the converter from Quadrant I (Rectifier) into Quadrant II/IV (Inversion mode)[cite: 1].
  • Ripple Frequency: Since there are 3 pulses per supply power loop cycle, the dominant output voltage harmonic ripple frequency occurs at $3f_s$ ($3 \times 50\text{Hz} = 150\text{Hz}$).
